## 💡 關於

**Deepseek Omnibox** 是一个轻量级、易于使用的扩展，可以从 DeepSeek AI 获取答案。

- **易于使用 —** 只需在地址栏 + 查询中输入 `@deepseek`
- **超轻量 —** < 100 KB
- **多浏览器支持 —** 支持所有 Chromium 浏览器（例如 Chrome, Edge, Opera）
- **注重隐私 —** 不收集任何个人数据
